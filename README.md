# Java Basic 📘☕️

이 저장소는 제가 Java 기초 개념을 학습하며 정리한 코드 예제들을 담고 있습니다.  
직접 코드를 작성하고, 이해한 내용을 정리하면서 학습했습니다.

## 📚 사용한 자료
- 온라인 강의 / 공식 문서 / 블로그 자료 등 참고
- Java의 기본 문법부터 클래스, 예외 처리, 파일 입출력, 객체 직렬화까지 다룹니다.
